# 🎵 Music Player (Electron App)

A simple and modern desktop music player built with Electron, HTML, CSS, and JavaScript, inspired by Spotify-like UI design.

It allows you to play local music files with a clean interface, custom controls, and smooth user experience.

⸻

## ✨ Features

* ▶️ Play / Pause music
* ⏭️ Next / Previous track
* 📊 Live progress bar with seek support
* 🎨 Custom UI with album covers
* 🎧 Auto-play next song when one finishes
* 🖥️ Desktop app built with Electron
* 🎵 Local music library support

⸻

## 🛠️ Tech Stack

* Electron
* JavaScript 
* HTML5 
* CSS3

## 🚀 How to Run the Project

1. Clone the repository
```bash
git clone https://github.com/rosros98/music-player-electron.git
cd music-player-electron
```

2. Install dependencies
```bash
npm install
```

3. Run the app
```bash
npm start
```
